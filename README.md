## Getting started

```sh
# 0. Clone the repo
git clone https://github.com/gabepereira/node-chat node-chat

# 1. Navigate to the folder and install dependencies
cd node-chat
npm install

# 3. Start the server and go to http://localhost:3001)
npm start

```

### Commands
